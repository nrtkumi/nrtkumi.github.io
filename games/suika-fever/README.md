# スイカ★フィーバー (SUIKA FEVER)

同じ果物を合体させてスイカを目指すマージパズルに、パチンコ・パチスロ風の過剰演出(保留オーラ・カットイン・リーチ・スロット・FEVER)を全部乗せした「脳汁」特化スイカゲーム。

- **プレイURL**: https://nrtkumi.github.io/games/suika-fever/
- **対応環境**: スマホ(縦画面・タッチ)/ PC(マウス)。依存ライブラリなしの単一HTML。

## 遊び方

- ドラッグ(またはマウス移動)で落下位置を照準。指を離す/クリックで即落下
- デフォルトで**オート落下ON**(0.3秒ごとに自動落下)。ボタンでOFFにできる
- 同じ果物どうしが触れると合体して1段階大きくなる: さくらんぼ → いちご → ぶどう → デコポン → かき → りんご → なし → もも → パイン → メロン → **スイカ**
- 果物が上の点線ラインを超えたままになるとゲームオーバー
- 得点 = 果物の基礎点 × コンボ × 倍率(オーラ/FEVER)× 100。**2.3秒以内に連続合体**するとコンボが伸びる

### 保留オーラ

落ちてくる果物にはパチンコの保留玉風の「オーラ」がランダムに付く。合体時は高い方のオーラを継承し、落下中にまれに**保留昇格**することもある。

| オーラ | 倍率 | 出現率 |
|---|---|---|
| 青保留 | ×1.2 | 22% |
| 緑保留 | ×1.5 | 13% |
| 赤保留 | ×2 | 8% |
| 金保留 | ×3 | 3.8% |
| 虹保留(プレミア) | ×5 | 1.2% |

### FEVER TIME と GOD RUSH

- 合体するたびFEVERゲージが溜まり、満タンで**FEVER TIME**(全得点2倍、約10秒、合体で延長)
- FEVER中にもう一度満タンにすると**GOD RUSH**(全得点4倍、8秒)

### 演出いろいろ

- **カットイン**: 連鎖数・果物ランクに応じて「激アツ!!」「確変!?」「雷神モード!!(落雷の嵐)」「神連鎖!!!」など
- **リーチ**: 大型果物(なし以上)のペアが接近すると「接近!!」ハイライト+鼓動音
- **スロット**: パイン合体で50%の確率で激熱リーチ(当たれば+77,700点&金保留)、スイカ完成で大当たりスロット
- **群予告**: まれに鳥の群れが横切り、次の果物が金保留確定
- ジャックポット時は紙吹雪・小判・花火・「祝」の文字・弾幕コメント(「脳汁!!」「ドーパミン!!」)が乱舞

## 技術構成

- 依存ライブラリなしの単一HTML(canvas 2D + Pointer Events)
- 効果音はWeb Audioでリアルタイム合成(オシレーター+ノイズ。ファンファーレ、ライザー、落雷など)
- **iOS消音スイッチ対策**: 初回タップ時に無音WAV(JSでArrayBuffer生成→Blob URL)を`<audio>`要素でループ再生してオーディオセッションを再生カテゴリへ切り替え、`AudioContext.resume()`のPromise完了を待ってアンロック判定、`onstatechange`で中断から自動復帰
- **2Dコンテキスト復帰対策**: iOS Safariの長時間セッションでcanvasの2Dコンテキストがリセットされても描画が崩れないよう、dprトランスフォームを毎フレーム再設定
- `prefers-reduced-motion` 対応: 画面シェイク・色収差・レーザー・色反転などの激しい演出を抑制
- OGP画像はゲームプレイ中のスクリーンショット(1200x630、`assets/ogp.jpg`)
