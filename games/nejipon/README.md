# ねじポン — ブリキ工房のネジ仕分けパズル

## 概要

色とりどりのネジをタップしてツールボックスへ仕分けるパズルゲーム。板が重なった部分のネジは外せない「ロック」状態になるため、**取り外す順番**を読み解くことが攻略の鍵。3本そろうとポップ！ステージクリアで強化を選ぶローグライト要素も搭載。テーマパークの待ち時間など「ちょっとした隙間」に最適な、じっくり考えて指一本で完結するパズルです。

## 遊び方・ルール

1. **タップ**：盤面に刺さったネジをタップすると、マッチする色のツールボックスへ飛んでいく
2. **ロック**：上に別の板が重なっているネジは灰色でロック表示。上の板のネジを先に外すと解除される
3. **ツールボックス**：同じ色のネジが3本そろうと **ポップ！** してリセット。空きスロットには最初に入った色が割り当てられる
4. **ホールディングトレイ**：対応するボックスがないネジは一時的にトレイへ退避できる（上限あり）
5. **クリア条件**：盤面のネジを全部外す
6. **ゲームオーバー**：ネジを置く場所（ボックスもトレイも）が満杯の状態でタップすると終了

### アップグレード一覧

ステージクリア後、3択から1つ選んで強化できます。

| アップグレード | 効果 |
|---|---|
| +1 トレイスロット | ホールディングトレイの容量を1増やす |
| +1 ツールボックス | アクティブな工具箱を1つ追加する |
| カラーセンス | タップした色と同じネジを全部ハイライト |
| アンドゥ | 直前の操作を1回取り消せるチャージを獲得 |
| マグネット | マッチする色のネジを自動収集する |

## 作成の背景

2025年後半、ハイブリッドカジュアルゲーム市場で最も伸長したのが「決定論的タップソート」系パズル（ネジ外し・ボルト整理・色仕分け）です。AppMagicの調査によれば、Q3 2025のハイブリッドカジュアルTop10のうち複数がこのカテゴリを占めており（[AppMagic Q3 Hybrid 2025](https://appmagic.rocks/blog/q3hybrid2025)）、Gamigionのランキングでも同様のトレンドが確認できます（[Top 10 HybridCasual Q3 2025](https://www.gamigion.com/top-10-hybridcasual-games-in-q3-2025/)）。

このジャンルの強みは「タイムプレッシャーなし」「完全な情報公開」「1手の重み」が組み合わさった純粋なパズル体験にあります。テーマパークやカフェの待ち時間など、途中で中断しても再開しやすい点も支持される理由です。

画面レイアウトは **サム・ゾーン設計**（[Smashing Magazine: Design for One-Hand Usage](https://www.smashingmagazine.com/2020/02/design-mobile-apps-one-hand-usage/)）に準拠し、ツールボックス・ホールディングトレイを画面下半分（親指が届きやすい領域）に配置しました。

## 技術構成

- **単一 HTML ファイル**：CSS・JS・すべてのアセットをインライン化
- **プレーン Canvas 2D**：ライブラリなし、外部画像ファイルなし
- **iOS オーディオアンロック**：silent WAV + WebAudio AudioContext パターン（ミュートスイッチ対策）
- **DPR 対応**：毎フレーム `ctx.setTransform(DPR,0,0,DPR,0,0)` を再適用、Retina/高密度ディスプレイで鮮明描画
- **Pointer Events 入力**：マウス・タッチ統一（click に依存しない）
- **PCワイド対応**：`width: min(100vw, calc(100dvh*0.66), 480px)` の縦長ポートレートカラムを中央配置
- **デザインコンセプト**：「ブリキのおもちゃ工房 / 青焼き設計図」— navy 背景にグリッド線、真鍮ネジ（フィリップスヘッド）、白破線のプレート、`Yusei Magic` + `Space Mono` フォント
