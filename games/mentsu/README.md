# 面子 — MENTSU 麻雀ソリティア

積まれた牌のうち**自由な牌**(上に何も乗っておらず、左右どちらかが空いている牌)を、同じ絵柄2枚タップで消していく麻雀ソリティア。**役**を作れば倍率、**時間**と戦い、**強化**を引いて何局も巡るローグライト。

- プレイ: https://nrtkumi.github.io/games/mentsu/
- 対応: スマホ(タッチ)/PC(S=洗牌, H=ヒント, U=待った, Esc=選択解除)。縦画面・単一HTML。

## 遊び方

1. **自由な牌**=上に牌が乗っておらず、左か右が空いている牌。タップで持ち上げ、同じ絵柄の自由牌をもう一枚タップで取り除く。
2. すべて消せばその局は和了(クリア)。
3. 続けて取ると**連(コンボ)**で倍率が上がる。手早く取るほど高得点。
4. **役**で大きなボーナス。時間が尽きるか、手詰まり(洗牌切れ)で仕事じまい。

### 役

| 役 | 条件 | ボーナス |
|----|------|----------|
| 役牌 | 三元牌(中・發・白)の対子を取る | +60 |
| 三色 | 直近で萬子・筒子・索子を出し揃える | +90 |
| 清一 | 直近3対子を同じ種類でそろえる | +70 |
| 断么九 | 直近3対子を2〜8の数牌だけでそろえる | +50 |
| ドラ | 印の付いた対子(各局1組) | 得点 ×3 |

### 強化(お守り)

延刻(+20秒) / 目利き(ヒント+2) / 洗牌符(洗牌+1) / 待った(アンドゥ+2) / 連風(コンボ持続+0.8秒) / 砂時計(時間の減りが遅くなる)

## 作成の背景

2026年の市場調査(deep-research で出典付き検証)で、**麻雀ソリティア**が急伸ジャンルだと判明。Vita Mahjong は2025年に世界DL7位、麻雀ソリティア・カテゴリは前年比 +189% で 2.58 億インストール。US Google Play 無料ゲーム上位の大半が1人用パズル/マッチ/ソートで占められている。当サイトに麻雀ソリティアは無かったため、定番の「自由牌マッチ」に**役による倍率・時間制・ローグライトの強化ドラフト**を足して差別化した。

- 麻雀ソリティア急伸(+189% YoY): https://www.deconstructoroffun.com/blog/2026/2/2/state-of-mobile-2026
- US Google Play 無料がパズル/マッチ優勢: https://www.appbrain.com/stats/google-play-rankings/top_free/game/us

## デザインコンセプト

**昭和の雀荘を上質に**。羅紗(felt)の深緑のテーブル、象牙色の牌、種類ごとに色分けした絵柄(萬子=朱、筒子=藍、索子=緑、字牌=墨)。牌は層ごとに左上へずらして影を落とし、積み重なりを立体的に表現。フォントは見出しに Shippori Mincho。生成AI的な紫グラデ・ガラス質感・ネオンは排した。

## 技術構成

- 単一HTML・依存ライブラリなし。canvas 2D で牌を手続き描画(萬子の漢数字＋萬、筒子の点、索子の竹、字牌の漢字)。層を後ろから前へ描画し、上層を左上にオフセットして3D表現。
- **解ける配置の保証**: 完成形(全牌が乗った状態)から「今フリーな2枚」を繰り返し取り除く逆順構築で出題。取り除きは他牌を必ず自由にするので、構築の除去順がそのまま正解手順になり、**必ず解ける**。同じ仕組みを洗牌(残り牌の再配置)・手詰まり検出・E2Eの自動クリアに再利用。
- iOS消音スイッチ対策、ダブルタップズーム無効化、セーフエリア対応、PCワイド画面で縦カラムに制限。効果音はWeb Audioのリアルタイム合成(残り時間が少ないと警告音)。
- 得点/最高記録は localStorage に保存。

## 実施したテスト

- インラインJSを抽出し `node --check`(構文OK)。
- 実ブラウザE2E(puppeteer-core + ローカルChrome, iPhoneビューポート/タッチ): 同梱の正解手順で**3局を実際に解ききり**、和了→強化ドラフト→次局の進行、コンボ・役・時間ボーナス、pageerror ゼロ、canvas描画、スクリーンショット目視を確認。`file://` ではなくローカルHTTPサーバ経由で実施。

## 教訓

- 麻雀ソリティアは「フリー判定」と「必ず解ける配置生成」が肝。**逆順構築**(完成形からフリー2枚を抜く)が両方を一度に解決し、洗牌・ヒント・自動テストまで同じコードで賄えた。
- 牌のフリー状態(取れる/取れない)を陰影で示し、選択中・ドラを縁取りで強調すると、積み重なった盤面でも操作が直感的になる。
