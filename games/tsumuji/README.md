# つむじ — 吸い込め、でっかくなれ！

お祭り広場を舞うつむじ風を操って、落ち葉・ゴミ・屋台をぜーんぶ吸い込む60秒アーケード。小さいものから順番に食べて大きくなり、最後は鳥居まで丸ごと吸い込め！

## プレイURL

<https://nrtkumi.github.io/games/tsumuji/>

## 対応環境

- スマートフォン (iOS Safari / Android Chrome) — タッチ操作
- PC ブラウザ (Chrome / Firefox / Safari) — WASD / 矢印キー
- 単一HTMLファイル、インストール不要

## 遊び方・ルール

### 基本操作

| 環境 | 操作 |
|------|------|
| スマホ | 画面をドラッグ → つむじ風がその方向へ移動 |
| PC | WASD または ←↑↓→ キー |

### ゲームの流れ

1. **あそぶ！** を押すと60秒カウントダウン開始
2. 自分のサイズより **小さいもの** に近づくと吸い込める
3. 吸い込むたびに少し大きくなり、より大きなものが食べられる
4. 素早く連続して吸い込むと **コンボボーナス** (x2, x3…) が加算
5. 時間切れでラウンド終了。スコアに応じてコインがもらえる

### 吸い込めるもの（小→大）

落ち葉 → 紙コップ → ポップコーン → 風船 → 傘 → ベンチ → ゴミ箱 → カート → 提灯屋台 → テント → 木 → 鳥居

### 強化ショップ

ラウンドで稼いだコインを使って永続強化を購入できる。

|----------|------|--------|
| 🌀 開始サイズ+ | 最初のつむじ風が大きくなる (+6px × Lv) | 4 |
| ⏱ 制限時間+ | 1ラウンド10秒延長 | 3 |
| ⭕ 吸い込み半径+ | 吸引できる距離が広がる (+20% × Lv) | 4 |
| 📈 成長率+ | 吸い込むたびの成長量が増える (+15% × Lv) | 4 |

## 作成の背景

### 2026年のモバイルゲームトレンドと本作の立ち位置

2025〜2026年にかけて、モバイルゲーム市場では **「アーケードコア＋メタ進行」ハイブリッド** が席巻している。60秒前後の短いラウンドを高リプレイ性で繰り返し、ラウンド外のコイン強化が長期エンゲージメントを支える構造だ。hole.io に代表される「.io型吸収アーケード」は参入障壁の低さとスケールの爽快感から2020年代を通じて根強い人気を保ち、2025年も上位カテゴリを維持している ([Mobile Games 2025 — MAF.ad](https://maf.ad/en/blog/top-mobile-games-2025/))。

ハイパーカジュアルが「5秒で学べる単純操作」から「意味のある強化と自分の成長実感」を付加した hybridcasual へ進化した点も注目されており、メタ進行の追加がリテンション率を大幅に改善する事例が相次いでいる ([Hyper-Casual Growth 2025 — StudioKrew](https://studiokrew.com/blog/how-hyper-casual-games-are-driving-growth-in-mobile-game-industry-2025/))。

つむじは「お祭り・公園」という日常的な舞台をつむじ風（竜巻）で席巻するファンタジーで差別化し、待ち時間（テーマパークの列など）でも楽しめる手軽さを意識した。操作設計は **サム・ゾーン理論**（親指が自然に届く画面下部にアクションを集約）に準拠し、ドラッグ先をつむじが追う方式で指がゲーム画面を隠さないよう配慮している ([Designing for One-Hand Usage — Smashing Magazine](https://www.smashingmagazine.com/2020/02/design-mobile-apps-one-hand-usage/))。

### デザインコンセプト

**「晴れた夏祭りの公園、俯瞰の地面と渦巻く風」**  
暖色の砂/芝パレット、フェス提灯の点光源ドット、絵文字オブジェクトが転がる祭り感。Zen Maru Gothic の丸みでポップさを出し、AI 御用達の紫グラデーション/グラスモーフィズムを完全排除した。

## 技術構成

- **単一 HTML ファイル** (CSS・JS・フォントリンクをすべてインライン)
- **Canvas 2D API** — 背景は初回のみオフスクリーンcanvasに一括生成しキャッシュ
- **Web Audio API** — SFXはすべてリアルタイム合成（外部音源ゼロ）
- **iOS 音声対策** — 無音WAV を `<audio loop>` でループ再生してオーディオセッションを再生カテゴリへ切り替え、消音スイッチ下でも AudioContext を動作させる
- **iOS ダブルタップズーム防止** — `touchend` 350ms判定 + `dblclick` / `gesturestart` preventDefault の3点セット
- **DPR 毎フレーム適用** — `ctx.setTransform(DPR,0,0,DPR,0,0)` を rAF の最初に呼ぶことで iOS のコンテキスト喪失後も描画を維持
- **Pointer Events 統一** — click 非依存（上記ダブルタップ対策と両立）
- **PC ワイド対応** — `width:min(100vw, calc(100dvh*0.56), 420px)` で縦長カラムに収め、キーボード (WASD/矢印) も対応
- **永続化** — `localStorage` (try/catch フォールバック)
- **Google Fonts** — Zen Maru Gothic (CDN)

### 教訓

- **ヘッドレステスト**: `"use strict"` スコープでは変数がモジュールローカルになるため、`Function()` コンストラクタで評価し戻り値で参照を取り出す方式が有効
- **rAF ループ初期化**: ループが参照する全変数を宣言時に初期化しないと title 画面の最初のフレームで例外が出てループが死ぬ（毎回の教訓）
- **OGP 撮影**: puppeteer は PNG しか出力しないため、`.jpg` 拡張子でも中身は PNG になる。ブラウザはどちらも正常に読む
- **favicon.ico 404**: ローカルサーバで E2E テストすると favicon.ico の 404 が console.error に出るが、ゲーム動作には無関係。フィルタリングして無視してよい
