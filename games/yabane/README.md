# 矢羽根

グリッドに並んだ矢印を順番にタップして盤外へ射出し、全部消すパズルゲーム。  
進路に別の矢があると発射不可 — 外側から解く依存パズル。ステージを重ねるほど盤が大きくなるローグライト。

## プレイURL

<https://nrtkumi.github.io/games/yabane/>

## 遊び方

| 操作 | 説明 |
|------|------|
| 矢印セルをタップ | 向いている方向へ発射。進路が全空なら成功、詰まっていれば失敗（シェイク） |
| ← → ↑ ↓ キー | カーソル移動（PC） |
| Enter / Space | カーソルのセルを発射（PC） |
| Z キー | アンドゥ（戻しチャージ消費） |
| W キー | 風モード切替（風チャージ消費） |
| スコア欄タップ | アンドゥ（スマホ） |
| コンボ表示タップ | 風モード切替（スマホ） |

### コンボ

1.2 秒（静心強化で延長）以内に続けて発射するとコンボが積み重なり、得点倍率が上がる。

### ステージ進行

- 4×4 からスタート、2ステージごとに 1 サイズ大きくなり最大 6×6。
- 全矢印を消すとステージクリア → お守りを 1 つ選択 → 次のステージへ。
- 残りの矢が全て詰まり（ソフトロック）、風チャージもない場合は終局。

## 強化一覧（お守り）

| 名前 | 効果 |
|------|------|
| 風（かぜ） | チャージ+1。使うと詰まった矢を 1 本強制発射（進路無視） |
| 戻し | アンドゥチャージ+1。直前の発射を 1 回取り消し |
| 慧眼（けいがん） | ステージ開始時、今すぐ撃てる矢を 3 秒間ハイライト |
| 二の矢 | 同じ向きの矢を連続して撃つと得点 ×2 |
| 的中 | 矢 1 本あたりの基本得点 +5 |
| 静心（せいしん） | コンボ継続ウィンドウ +0.6 秒 |

## 作成の背景

2026年5月、矢印を滑らせてゴールを目指す系パズル「Arrows Puzzle Escape」が App Store / Google Play で世界ダウンロード 1 位を記録し、矢印系スライドパズルが急上昇トレンドとなった。本作はその「向きと進路の依存」を 1 人用ブラウザゲーム向けに翻案し、弓道の世界観とローグライトの強化選択を組み合わせたもの。

## デザインコンセプト

**弓道・和弓の美意識** — 生成り（オフホワイト和紙）地に墨黒の矢、朱（#c8332b）と金（#c8a24a）のアクセント。矢印は鏃と矢羽根を持つ「矢」として描き、的（まと）の同心円モチーフを盤の背景に配置。

## 技術構成

| 項目 | 内容 |
|------|------|
| 実装 | 単一 HTML（canvas 2D + Vanilla JS） |
| フォント | Yuji Syuku（見出し）、Zen Kaku Gothic New（UI） |
| 音声 | Web Audio API によるリアルタイム合成 |
| iOS 音声対策 | 無音 WAV ループ + AudioContext.resume() |
| 永続化 | localStorage（キープレフィックス: yabane_） |
| 盤面生成 | ランダム配置 → 貪欲ソルバーで解答可能性検証、最大 200 試行 |
| 入力 | Pointer Events（タッチ / マウス統一）+ キーボード対応 |
